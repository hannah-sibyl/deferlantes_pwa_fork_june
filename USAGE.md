# Regatta Logger — Guide d'utilisation

## Avant de partir en mer

1. **Ouvrir l'application une fois avec une connexion wifi.**
   Cela installe le cache hors-ligne. Une seule fois suffit.

2. **Ne pas recharger la page pendant la course.**
   Si la page est rechargée, la sauvegarde automatique devra être réactivée.

---

## Pendant la course

### Enregistrer un changement de configuration
Appuyer sur le bouton correspondant à la nouvelle configuration.
Chaque appui est horodaté et apparaît dans l'historique en bas de page.

### Corriger l'heure d'un changement (appui long)
Si un changement a été effectué mais non enregistré dans l'instant :
- Maintenir le bouton appuyé pendant **1,5 seconde**
- Une fenêtre s'ouvre avec un menu déroulant
- Sélectionner le décalage à soustraire (ex. − 5 min)
- Appuyer sur **Sauvegarder** pour enregistrer avec l'heure corrigée
- Appuyer sur **Annuler** pour ignorer

---

## Sauvegarde automatique (optionnel)

- Activer le toggle **"Activer export csv automatique"** en bas de l'écran
- Une fenêtre de l'OS s'ouvre pour choisir l'emplacement et le nom du fichier
- Une fois activé, le fichier CSV est écrasé automatiquement toutes les **6 heures**
- Peut être activé hors-ligne (pas besoin de wifi)
- **Si la page est rechargée**, réactiver le toggle pour reprendre la sauvegarde automatique

---

## Export manuel

Appuyer sur **Export CSV** en bas de l'écran à tout moment pour télécharger le fichier de la course.

---

## Réinitialiser la course

Appuyer sur **Reset** — une confirmation est demandée avant d'effacer toutes les données.

---

## Notes

- Les données sont sauvegardées automatiquement dans le navigateur à chaque appui.
  Elles survivent à un rechargement de page.
- L'application fonctionne entièrement hors-ligne après la première ouverture avec wifi.
- Le thème clair/sombre se bascule avec le bouton 🌙 / ☀️ en haut à gauche.
